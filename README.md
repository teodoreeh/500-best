# 500 BEST

Рандомайзер топ-500 фильмов Letterboxd. Жмёшь «КРУТИТЬ» — выпадает случайный фильм, открываешь его на Letterboxd, собираешь коллекцию просмотренного. Полностью статичное приложение: ни сервера, ни оплаты.

## Быстрый старт

Открой `index.html` двойным кликом — всё работает офлайн (база и 3 демо-постера вшиты внутрь).

Для разработки удобнее поднять локальный сервер (чтобы работал `fetch` данных):

```bash
# любой из вариантов
python3 -m http.server 8000
# или
npx serve
```

Затем открой http://localhost:8000

## Разработка в Claude Code

В корне лежит `CLAUDE.md` — контекст проекта (архитектура, дизайн-язык, утверждённые решения, дорожная карта). Claude Code читает его сам. Начни с задачи №1 из дорожной карты.

## Деплой (бесплатно)

1. Залей папку в репозиторий GitHub.
2. Подключи репозиторий к **Cloudflare Pages** (или включи **GitHub Pages**). Билд не нужен — это статика.
3. (Опционально) Поддомен: смени DNS `films.masterofthestar.ru` с текущей A-записи (сервер Selectel) на CNAME к хостингу. После этого сервер Selectel можно отключить.

## Данные

`data/movies.json` — снапшот базы. Первоисточник, который автор обновляет сам:
`https://github.com/bigbadraj/Letterboxd-List-JSONs`

## Постеры

Постеры подтягиваются через TMDB разовым скриптом (`scripts/enrich-posters.mjs`). Продукт использует TMDB API, но не одобрен и не сертифицирован TMDB — не забудь про атрибуцию в интерфейсе.
